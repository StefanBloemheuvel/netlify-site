Profile


My PhD is focussed on using graph analysis on industrial / sensor data, under supervision of Dr. Habil. Martin Atzmueller.

In Mathematics, graph theory focusses on a collection of items connected by edges. With help of Graph Signal Processing and Graph Neural Networks, we can both use the numerical information from time-series data as well as the topological information that is apparent in the graph.
